# Exception Handling

This project demonstrates adding proper exception handling to Python code.  
It contains a file with broken code (`broken_code.py`) and the corrected version (`fixed_code.py`).  

## Skills Highlighted
- Handling runtime exceptions
- Adding try/except blocks
- Writing robust, fail-safe Python code

## How to Use
1. Run `broken_code.py` → Observe code crashing on errors.  
2. Run `fixed_code.py` → Corrected code runs safely and handles exceptions.
