# Syntax Errors

This project demonstrates fixing common Python syntax errors.  
It contains a file with broken code (`broken_code.py`) and the corrected version (`fixed_code.py`).  

## Skills Highlighted
- Python syntax correction
- Debugging basic errors
- Understanding code structure

## How to Use
1. Run `broken_code.py` → Observe syntax errors.  
2. Run `fixed_code.py` → Correct code runs successfully.
